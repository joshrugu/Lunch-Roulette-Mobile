Lunch Roulette

Files:
- index.html: page structure
- style.css: design and mobile layout
- app.js: spin logic
- foods.json: categories, foods, budgets, challenges, fortunes
- manifest.webmanifest + sw.js + icons: PWA support

How to update foods:
1. Open foods.json
2. Add or remove items inside the relevant category
3. Upload/commit changes to GitHub
4. GitHub Pages will update automatically

GitHub Pages:
Upload all files to the root of your repository.
Settings > Pages > Deploy from a branch > main > /root.
